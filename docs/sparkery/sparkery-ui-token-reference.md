# Sparkery UI Token Reference

## Color Tokens

- Background: `--sparkery-saas-bg`, `--sparkery-saas-bg-soft`, `--sparkery-saas-card`
- Text: `--sparkery-saas-text`, `--sparkery-saas-muted`
- Status: `--sparkery-saas-status-success`, `--sparkery-saas-status-warning`, `--sparkery-saas-status-danger`, `--sparkery-saas-status-info`
- Severity: `--sparkery-saas-severity-low`, `--sparkery-saas-severity-medium`, `--sparkery-saas-severity-high`

## Typography Tokens

- Headers: `--sparkery-saas-font-h1` ... `--sparkery-saas-font-h6`
- Body/Caption: `--sparkery-saas-font-body`, `--sparkery-saas-font-caption`
- Line heights: `--sparkery-saas-line-tight`, `--sparkery-saas-line-base`

## Spacing & Radius

- Spacing rhythm: `--sparkery-saas-space-1` ... `--sparkery-saas-space-5`
- Radius: `--sparkery-saas-radius-sm`, `--sparkery-saas-radius-md`, `--sparkery-saas-radius-lg`

## Elevation & Layering

- Shadows: `--sparkery-saas-shadow-soft`, `--sparkery-saas-shadow-soft-md`, `--sparkery-saas-shadow-soft-lg`
- Z-index: `--sparkery-saas-z-base`, `--sparkery-saas-z-dropdown`, `--sparkery-saas-z-modal`

## Density & Theme Runtime

- Density: `data-sparkery-density = comfortable | compact | dense`
- Theme: `data-sparkery-theme = light | dark-neutral`
- Control height token: `--sparkery-saas-control-height`
