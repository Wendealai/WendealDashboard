# Rednote Img Generator 快速启动指南

## 🚀 5分钟快速上手

### 步骤 1：启动应用

```bash
# 启动开发服务器
npm run dev

# 或使用生产模式
npm run build
npm run preview
```

### 步骤 2：访问功能

1. 打开浏览器访问应用
2. 点击侧边栏的 **Social Media** 菜单
3. 在工作流卡片中找到 **Rednote Img Generator**
4. 点击卡片进入功能页面

### 步骤 3：开始使用

1. **输入文案**

   ```
   在左侧文本框输入你的小红书文案
   例如：
   今天分享一个超实用的生活小技巧！
   让你的生活变得更加便利 ✨
   ```

2. **点击生成**
   - 点击"生成图片"按钮
   - 等待处理完成（通常 2-5 秒）

3. **查看结果**
   - 右侧自动显示生成的 HTML
   - 预览区域显示渲染效果

4. **使用内容**
   - 点击"新窗口打开"查看完整效果
   - 点击"复制HTML"获取源代码

## 🔧 n8n Webhook 配置

### 创建 Webhook 节点

在 n8n 中创建新工作流：

1. **添加 Webhook 节点**
   - URL: `/webhook/rednoteimggen`
   - Method: `POST`
   - Response Mode: `Last Node`

2. **处理请求数据**

   ```javascript
   // 提取文案内容
   const content = $json.body.content;

   // 生成 HTML（根据实际需求实现）
   const html = generateRednoteHTML(content);

   // 返回响应
   return {
     success: true,
     html: html,
     timestamp: new Date().toISOString(),
   };
   ```

3. **返回响应**
   - 确保返回包含 `html` 字段的 JSON

### 测试 Webhook

使用 curl 测试：

```bash
curl -X POST https://n8n.wendealai.com/webhook/rednoteimggen \
  -H "Content-Type: application/json" \
  -d '{
    "content": "测试文案内容",
    "timestamp": "2024-10-29T00:00:00.000Z"
  }'
```

预期响应：

```json
{
  "success": true,
  "html": "<!DOCTYPE html>...",
  "timestamp": "2024-10-29T00:00:00.000Z"
}
```

## ❓ 常见问题

### Q1: 生成失败怎么办？

**A:** 检查以下几点：

1. 网络连接是否正常
2. n8n webhook 服务是否运行
3. 文案内容是否为空
4. 查看浏览器控制台错误

### Q2: 新窗口打不开？

**A:** 可能是浏览器阻止了弹窗：

1. 检查浏览器弹窗设置
2. 允许当前网站弹窗
3. 或使用"复制HTML"功能

### Q3: HTML 显示异常？

**A:** 尝试以下方法：

1. 在新窗口打开查看
2. 检查 HTML 代码是否完整
3. 查看控制台错误信息

### Q4: 支持哪些格式？

**A:**

- 纯文本文案
- 支持 emoji 表情
- 支持特殊符号
- 最大 2000 字符
